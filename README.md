# KR260 ROS2 Setup Guide

This repository documents the process of installing ROS2 Humble on the AMD KR260 Robotics Starter Kit and verifying the installation using TurtleSim.

## Hardware

* KR260 Robotics Starter Kit
* Logitech C270 USB Camera

## Software

* Ubuntu 22.04
* ROS2 Humble

## Installation Script

The ROS2 installation script is located in:

scripts/install_ros2.sh

## Verification

After installing ROS2, TurtleSim can be used to verify that ROS2 is working correctly.

  
## Usage

1. Boot the KR260 board.
2. Open a terminal.
3. Enter the PYNQ environment.

Then run the installation script located in the `scripts` folder.

After installation, ROS2 can be verified using the TurtleSim simulator.
